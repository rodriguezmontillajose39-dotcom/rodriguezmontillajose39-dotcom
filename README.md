## Proyecto destacado

### [MoodFix](https://github.com/rodriguezmontillajose39-dotcom/Moodfix)
Sistema de recomendación de películas basado en el estado de ánimo del usuario. Combina un motor de filtrado determinístico sobre el catálogo de TMDB con inteligencia artificial (Claude Haiku) para generar sugerencias personalizadas y evitar la fatiga de decisión.

**Stack:** React · Vite · Flask · SQLite · TMDB API · Claude Haiku
